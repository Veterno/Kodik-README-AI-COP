# 🚀 kodik-readme-ai

## 📝 Описание

Автоматическая генерация README.md с использованием OpenAI-совместимых API и анализом контекста (локальный fallback). Утилита экономит время разработчиков, создавая безопасную и качественную документацию.

## ✨ Ключевые возможности

Вот перевод текста на русский язык:

- **AI-генерация** — Генерация README с помощью OpenAI-совместимых API (Ollama, LM Studio, Groq, DeepSeek и др.)
- **Контекстный анализ** — Сбор контекста из кода, Git-логов, документации и интерактивного опроса
- **Многоязычность** — Поддержка генерации на разных языках и автоматический перевод разделов
- **Детектор стека** — Автоматическое определение языка, фреймворка, менеджера пакетов и дополнительных технологий
- **Интерактивный режим** — CLI-опрос для уточнения аудитории, тона и бизнес-ценности
- **Гибкая конфигурация** — Настройка через CLI-аргументы, файл конфигурации (JSON/YAML) и переменные окружения
- **Валидация** — Проверка сгенерированного README с помощью LLM-as-a-Judge
- **Безопасность** — Маскировка чувствительных данных (ключей, паролей) в выводе
- **Бенчмаркинг** — Скрипт для замера производительности генерации

## 🛠️ Стек технологий

- **Язык:** Node.js (JavaScript)
- **Пакетный менеджер:** npm
- **Дополнительно:** axios, dotenv, js-yaml, prompts, yargs, jest

## 📦 Быстрый старт

### Требования

- Node.js >= 14

### Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск
npm start -- .
npx kodik-readme-ai .
```
