# 🚀 Kodik README AI

## 📝 Описание

Автоматический генератор README.md с использованием OpenAI-совместимых API и локальным fallback. Анализирует код, манифесты, Git-историю и бизнес-контекст для создания качественной документации. Экономит время разработчиков, автоматизируя создание и перевод документации.

## ✨ Ключевые возможности

- **AI-генерация** — Генерация README с помощью OpenAI-совместимых API (OpenAI, DeepSeek, Groq, Ollama, LM Studio).
- **Контекстный анализ** — Сбор контекста из исходных файлов, манифестов, Git-логов и бизнес-документации.
- **Многоязычность и перевод** — Автоматический перевод разделов README на целевой язык с сохранением форматирования Markdown.
- **Детектор стека** — Автоматическое определение языка программирования, фреймворка и менеджера пакетов по манифестам и файлам.
- **Интерактивный режим** — CLI-опрос для уточнения аудитории, тона, бизнес-ценности и ключевых функций.
- **Гибкая конфигурация** — Настройка через CLI-аргументы, файл конфигурации (JSON/YAML) и переменные окружения.
- **Валидация** — LLM-as-a-Judge валидация сгенерированного README на соответствие проекту.
- **Бенчмаркинг** — Скрипты для бенчмаркинга качества генерации на эталонных проектах.

## 🛠️ Стек технологий

- **Язык:** Node.js (JavaScript)
- **Пакетный менеджер:** npm
- **Дополнительно:** axios, dotenv, js-yaml, prompts, yargs, jest

## 📦 Быстрый старт

### Требования

- Node.js >= 18

### Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск
npm start -- .
```
