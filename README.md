# 🚀 Kodik README AI

## 📝 Описание

Автоматический генератор README.md с использованием OpenAI-совместимых API и анализом контекста проекта. Поддерживает локальный fallback без AI, многоязычность, детекцию стека технологий, Git-контекст и гибкую конфигурацию через CLI или веб-интерфейс.

## ✨ Ключевые возможности

- **AI-генерация** — Генерация README с помощью OpenAI-совместимых API (OpenAI, DeepSeek, Groq, Ollama, LM Studio).
- **Контекстный анализ** — Сбор контекста из исходных файлов, манифестов, Git-логов и бизнес-документации.
- **Многоязычность** — Автоматический перевод README на целевой язык с сохранением форматирования Markdown.
- **Детектор стека** — Автоматическое определение языка программирования, фреймворка и менеджера пакетов по манифестам и файлам.
- **Интерактивный режим** — CLI-опрос для уточнения аудитории, тона и бизнес-ценности проекта.
- **Гибкая конфигурация** — Настройка через CLI-аргументы, файл конфигурации (JSON/YAML) и переменные окружения.
- **Валидация** — Локальная и AI-валидация сгенерированного README на наличие обязательных разделов и корректность ссылок.
- **Безопасность** — Маскировка чувствительных данных (API-ключи, пароли) в логах и сгенерированном README.

## 🛠️ Стек технологий

- **Язык:** Node.js (JavaScript)
- **Фреймворк:** Express
- **Пакетный менеджер:** npm
- **Дополнительно:** axios, ejs, highlight.js, marked, simple-git, jest

## 📦 Быстрый старт

### Требования

- Node.js >= 18

### Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск
npm start
```
