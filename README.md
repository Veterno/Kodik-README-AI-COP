# 🚀 Kodik-README-AI

## 📝 Описание

Утилита для автоматической генерации README.md с использованием OpenAI-совместимых API и анализа контекста проекта. Поддерживает локальную генерацию без AI, многоязычность, гибкую конфигурацию и плагины.

## ✨ Ключевые возможности

- **AI-генерация** — Генерация README с помощью OpenAI-совместимых API (Ollama, LM Studio, Groq, DeepSeek и др.) с поддержкой JSON-ответа.
- **Контекстный анализ** — Сбор контекста из исходных файлов, Git-лога, манифестов и документации для более точной генерации.
- **Многоязычность** — Поддержка русского и английского языков для интерфейса и генерации, а также автоматический перевод разделов README.
- **Детектор стека** — Автоматическое определение языка программирования, фреймворка, менеджера пакетов и дополнительных технологий по манифестам и файлам.
- **Интерактивный режим** — CLI-опрос для уточнения аудитории, тона, бизнес-ценности и ключевых функций проекта.
- **Гибкая конфигурация** — Настройка через CLI-аргументы, файл конфигурации (JSON/YAML), переменные окружения и плагины.
- **Валидация и автоисправление** — Локальная валидация разделов, ссылок и форматирования Markdown с возможностью автоматического исправления.
- **Плагинная архитектура** — Поддержка пользовательских плагинов для расширения функциональности через систему хуков.

## 🛠️ Стек технологий

- **Язык:** Node.js (JavaScript)
- **Пакетный менеджер:** npm
- **Дополнительно:** Express, EJS, Jest, Yargs, Axios, js-yaml, marked, highlight.js, simple-git, adm-zip, multer

## 📦 Быстрый старт

### Требования

- Node.js >= 18

### Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск
npm start
```
