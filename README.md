# 🚀 Kodik-README-AI-COP

## 📝 Описание

Проект представляет собой микросервис на **Node.js (JavaScript)**. Бизнес-ценность: Бизнес-цель репозитория Ollama — демократизация и упрощение локального использования искусственного интеллекта (ИИ) для разработчиков и бизнеса..

## ✨ Ключевые возможности

- ✨ Запуск моделей
- ✨ управление моделями
- ✨ REST API
- ✨ квантование моделей
- ✨ кастомизация через Modelfile
- ✨ кроссплатформенность
- ✨ аппаратное ускорение.

## 🛠️ Стек технологий

- **Язык:** Node.js (JavaScript)
- **Пакетный менеджер:** npm
- **Дополнительно:** Testing

### Зависимости
```
axios, dotenv, inquirer
```

### Dev-зависимости
```
jest
```

## 📦 Быстрый старт

### Требования

- Node.js v18 или новее
- npm (входит в состав Node.js)

### Установка и запуск

```bash
# 1. Установка зависимостей
npm install

# 2. Запуск
npm start
```
